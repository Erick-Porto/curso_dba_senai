Recursos valiosos para aprofundar seu conhecimento:

  • Documentação PostgreSQL:
    • Chapter 13. Transactions
    • 13.2. Transaction Isolation
    • 13.3. Explicit Locking
    • Chapter 13. Concurrency Control (MVCC)
  • MySQL (InnoDB):
    • 15.2. InnoDB and ACID Model
    • 15.7. InnoDB Transaction Model and Locking
    • 15.7.2.3 Transaction Isolation Levels
    • 15.7.5.2 Deadlock Detection
  • Livros Clássicos (Nível Avançado):
    • "Database System Concepts" por Silberschatz, Korth, and Sudarshan (Capítulos sobre Transações e Controle de Concorrência).
    • "Readings in Database Systems" (The Red Book) - Contém artigos seminais, incluindo "A Critique of ANSI SQL Isolation Levels" por Berenson, Bernstein, et al.
  • Artigos Online e Tutoriais:
    • Busque por: "SQL transaction isolation explained", "understanding database deadlocks", "optimistic vs pessimistic concurrency control", "MVCC explained".
    • Muitos blogs de tecnologia e SGBDs específicos (ex: Percona, Brent Ozar para SQL Server) têm excelentes posts.

(Lembre-se de que os links são sugestões e podem precisar de atualização. Sempre priorize a documentação oficial do seu SGBD.)
