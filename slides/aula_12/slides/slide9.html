Recapitulação Rápida H1

Pilares Fundamentais:

  • Uma Transação é uma unidade lógica de trabalho, crucial para manter a consistência dos dados.
  • Comandos chave: BEGIN TRANSACTION (ou START TRANSACTION) para iniciar.
  • COMMIT para tornar as alterações permanentes.
  • ROLLBACK para desfazer todas as alterações da transação.
  • O princípio "Tudo ou Nada" (Atomicidade) é central.
