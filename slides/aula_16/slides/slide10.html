Métricas Fundamentais (Sistema Operacional) - Parte 2

Disco / I.O. (Input/Output):

  • IOPS (Operações de I/O por segundo): Capacidade de leituras/escritas do disco.
  • Latência de Disco (ms): Tempo para completar uma operação de I/O. Valores altos indicam disco lento.
  • Throughput (MB/s): Volume de dados transferido por segundo.
  • Queue Length: Fila de espera por operações de I/O.

Rede:

  • Tráfego (Mbps/Gbps): Volume de dados entrando/saindo.
  • Conexões Abertas/Ativas: Número de conexões de rede.
  • Latência de Rede (ms): Atraso na comunicação.
  • Perda de Pacotes: Pode indicar problemas de conectividade.

Ferramentas comuns no Linux: iostat, netstat, ss. Windows: Monitor de Recursos.

O I/O de disco é frequentemente o maior gargalo para bancos de dados!
