Como Gerar Planos de Execução no MySQL

O comando principal para obter o plano de execução é EXPLAIN.

Sintaxe básica:

EXPLAIN SELECT col1, col2 FROM sua_tabela WHERE condicao;

Pode-se usar formatos diferentes para mais detalhes (especialmente em versões mais recentes do MySQL):

EXPLAIN FORMAT=JSON SELECT ... ;
EXPLAIN FORMAT=TREE SELECT ... ; (MySQL 8.0.16+)

No MySQL Workbench:

  • Escreva sua consulta no editor SQL.
  • Clique no ícone ("Explain current statement") na barra de ferramentas do editor SQL.
  • O Workbench exibirá o plano de execução em formato tabular e, mais importante, o Visual Explain Plan, que é uma representação gráfica muito intuitiva.

Para PostgreSQL (apenas como referência): EXPLAIN (ANALYZE, BUFFERS) SELECT ...;
