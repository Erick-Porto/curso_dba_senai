Ferramentas de Otimização no Ecossistema MySQL

Além do EXPLAIN, outras ferramentas ajudam na otimização:

  • SHOW PROFILE / SHOW PROFILES:

    Permite analisar o tempo gasto em cada estágio da execução de uma query específica. (Ativar com SET profiling = 1; na sessão).

  • Performance Schema:

    Coleta detalhada de estatísticas de desempenho em baixo nível (eventos de espera, estágios de query, locks, etc.). Pode ser complexo, mas muito poderoso. O MySQL Workbench utiliza dados do Performance Schema para alguns de seus relatórios.

  • Slow Query Log:

    Registra queries que demoram mais que long_query_time para executar. Essencial para identificar as queries mais problemáticas em produção.

  • Percona Toolkit (e.g., pt-query-digest):

    pt-query-digest é uma ferramenta externa fantástica para analisar e agregar informações do Slow Query Log, gerando relatórios detalhados sobre as queries mais custosas.

  • MySQL Workbench:

    Além do Visual Explain, oferece relatórios de desempenho e uma interface para explorar o Performance Schema.
