Ferramentas de Monitoramento Internas ao MySQL

O MySQL oferece comandos e esquemas internos valiosos para monitoramento:

Comandos SHOW STATUS:

  • SHOW GLOBAL STATUS;: Contadores globais do servidor (Uptime, Queries, Slow_queries, Threads_connected, etc.).
  • SHOW SESSION STATUS;: Contadores para a sessão atual.
  • Exemplo: SHOW GLOBAL STATUS LIKE 'Innodb_buffer_pool_read%';
  • No MySQL Workbench: Muitos desses status são exibidos na aba "Server Status".

INFORMATION_SCHEMA:

  • Um banco de dados que contém metadados sobre o SGBD.
  • Tabelas úteis:
    • PROCESSLIST: Sessões ativas e suas queries.
    • TABLES: Informações sobre tabelas (tamanho, engine).
    • STATISTICS: Informações sobre índices.
    • INNODB_TRX, INNODB_LOCKS, INNODB_LOCK_WAITS: Detalhes sobre transações e locks do InnoDB.
  • No MySQL Workbench: Acessível via editor SQL.
