Demonstração: Monitorando com MySQL Workbench

Vamos explorar juntos as capacidades de monitoramento do MySQL Workbench:

Painel "Server Status":

  • Visão geral da saúde do servidor.
  • Métricas chave de tráfego, conexões, uso de buffer pool do InnoDB.
  • Identificação rápida de atividade anormal.

Análise de "Performance Reports":

  • Relatórios visuais sobre I/O, locks, hotspots de tabelas.
  • Identificação de schemas/tabelas mais acessadas.

Executando Consultas no INFORMATION_SCHEMA:

  • Como usar o editor SQL para consultar PROCESSLIST.
  • Verificar locks com INNODB_LOCKS e INNODB_LOCK_WAITS.
  • Exemplo prático de identificação de uma query lenta em execução.

Contextualizando com DER (Diagrama Entidade-Relacionamento):

  • Breve visualização do DER do banco de exemplo (e.g., Sakila) para entender a estrutura dos dados que estamos monitorando/consultando.

Esta demonstração será interativa. Prepare seu MySQL Workbench!
