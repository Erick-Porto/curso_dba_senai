JOINs Múltiplos

Podemos encadear JOINs para combinar dados de três ou mais tabelas.

Exemplo: Ver nome do cliente, nome do produto e quantidade comprada.


SELECT
    c.Nome AS NomeCliente,
    pr.NomeProduto,
    ip.Quantidade
FROM
    Clientes c -- Começa com Clientes
INNER JOIN
    Pedidos p ON c.ClienteID = p.ClienteID -- Junta com Pedidos
INNER JOIN
    ItensPedido ip ON p.PedidoID = ip.PedidoID -- Junta com ItensPedido
INNER JOIN
    Produtos pr ON ip.ProdutoID = pr.ProdutoID -- Junta com Produtos
WHERE
    c.Cidade = 'São Paulo'; -- Pode adicionar filtros normalmente

A ordem dos INNER JOINs geralmente não afeta o resultado, mas pode afetar a performance.
