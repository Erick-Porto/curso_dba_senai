Subqueries (Subconsultas) - O Que São?

Uma consulta SELECT completa aninhada (embutida) dentro de outra instrução SQL (SELECT, INSERT, UPDATE, DELETE).

A subquery (consulta interna) é executada primeiro, e seu resultado é usado pela consulta externa.

Analogia: Fazer uma pergunta menor ("Qual a média de preço?") para poder responder a uma pergunta maior ("Quais produtos estão acima da média?").


SELECT NomeProduto FROM Produtos
WHERE Preco > (SELECT AVG(Preco) FROM Produtos); -- Subquery entre parênteses
                     ----------------------------
                           Consulta Interna
--------------------------------------------------
                      Consulta Externa
