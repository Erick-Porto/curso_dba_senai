UPDATE - Modificando Dados

Para alterar valores em linhas existentes:


UPDATE nome_tabela
SET coluna1 = 'novo_valor1',      -- Define novos valores
    coluna_numerica = coluna_numerica + 10,
    outra_coluna = outra_coluna * 0.9
WHERE condicao; -- <<< ESSENCIAL! Filtra quais linhas serão atualizadas

PERIGO ABSOLUTO!

Omitir a cláusula WHERE no UPDATE fará com que TODAS AS LINHAS da tabela sejam atualizadas com os valores definidos no SET!

-- NÃO FAÇA ISSO SEM CERTEZA ABSOLUTA!
UPDATE Produtos SET Preco = 0; -- Zera o preço de TUDO!

Exemplo Seguro: Atualizar o email de um cliente específico.


UPDATE Clientes
SET Email = 'novo.email.ana@email.com'
WHERE ClienteID = 123; -- Atualiza APENAS o cliente com ID 123
