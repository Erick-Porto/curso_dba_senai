INSERT INTO - Adicionando Dados

Para inserir uma nova linha:

Especificando Colunas (Recomendado):


INSERT INTO Clientes (Nome, Email, Cidade, Idade) -- Lista de colunas
VALUES ('Ana Banana', 'ana.b@email.com', 'Recife', 25); -- Valores na mesma ordem

Omitindo Colunas (Assume todas, na ordem da tabela - CUIDADO!):


INSERT INTO Clientes -- Ordem dos valores DEVE bater com ordem das colunas na tabela!
VALUES (DEFAULT, 'Beto Carrero', 'beto.c@email.com', 'Penha', 50, '2024-08-01'); -- Ex: ID pode ser DEFAULT se auto-incremento
