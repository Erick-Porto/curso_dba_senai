Índices: Visualização no MySQL

É essencial saber como verificar os índices existentes em suas tabelas:

Comando SHOW INDEX FROM nome_da_tabela;

Este é o comando SQL principal para listar todos os índices de uma tabela específica.

SHOW INDEX FROM sakila.film;

Principais Colunas do Resultado:

ColunaDescrição
TableNome da tabela.
Non_unique0 se o índice é único, 1 caso contrário.
Key_nameNome do índice (ex: PRIMARY, idx_title, ou o nome da constraint UNIQUE).
Seq_in_indexNúmero sequencial da coluna dentro de um índice composto (1 para a primeira coluna, 2 para a segunda, etc.).
Column_nameNome da coluna que faz parte do índice.
CollationCollation da coluna no índice ('A' para ascendente, NULL para não ordenado/hash).
CardinalityEstimativa do número de valores únicos no índice. Crucial para o otimizador! Valores altos geralmente indicam boa seletividade.
Sub_partComprimento do prefixo indexado para colunas string/blob, ou NULL.
PackedIndica como a chave está empacotada (NULL se não estiver).
Null'YES' se a coluna pode conter NULLs, '' (vazio) se não.
Index_typeTipo de estrutura do índice (ex: BTREE, FULLTEXT, HASH, RTREE).
CommentComentários sobre o índice (ex: "primary key", "disabled").
Index_commentComentário fornecido ao criar o índice com a cláusula COMMENT.

MySQL Workbench:

  • Navegador de Schema (Schema Navigator):
    • Expanda o schema (banco de dados).
    • Expanda a tabela desejada.
    • Expanda a seção "Indexes". Lá você verá a lista de índices.
  • Table Inspector:
    • Clique com o botão direito na tabela no Navegador de Schema.
    • Selecione "Table Inspector".
    • Vá para a aba "Indexes".

Demonstração Rápida: Executar SHOW INDEX FROM sakila.actor; no Workbench e inspecionar os índices da tabela `actor` usando o Table Inspector.
