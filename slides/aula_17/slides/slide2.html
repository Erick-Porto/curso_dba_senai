Objetivos da Aula

Ao final desta aula, você será capaz de:

  • Compreender a importância da otimização de consultas para o desempenho do banco de dados.
  • Entender o papel das estatísticas do banco de dados no processo de otimização.
  • Aprender a gerar e interpretar planos de execução de consultas usando EXPLAIN no MySQL Workbench.
  • Dominar os conceitos fundamentais sobre índices (estrutura, tipos, impacto).
  • Aplicar técnicas de otimização de consultas SQL através de exercícios práticos.
  • Identificar oportunidades para uso de estratégias avançadas de indexação.
