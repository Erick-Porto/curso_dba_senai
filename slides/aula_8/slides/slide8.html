Finalidade da Linguagem SQL

SQL é dividida em subconjuntos de comandos:

  • DQL (Data Query Language): Consultar dados (SELECT). <- FOCO HOJE!
  • DML (Data Manipulation Language): Inserir (INSERT), Atualizar (UPDATE), Deletar (DELETE).
  • DDL (Data Definition Language): Criar (CREATE), Alterar (ALTER), Remover (DROP) estruturas.
  • DCL (Data Control Language): Gerenciar permissões (GRANT, REVOKE).
  • TCL (Transaction Control Language): Gerenciar transações (COMMIT, ROLLBACK).
