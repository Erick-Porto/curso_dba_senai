Limitando Resultados com LIMIT

Às vezes, não precisamos de todas as linhas que atendem aos critérios, apenas de uma amostra ou das primeiras.

A cláusula LIMIT restringe o número de linhas retornadas pela consulta.

Posição: Geralmente, vem depois do ORDER BY (se houver).


SELECT colunas
FROM tabela
[WHERE condicao]
[ORDER BY coluna]
LIMIT numero_de_linhas;

Exemplo: Buscar os 5 produtos mais caros.


SELECT Nome, Preco
FROM Produtos
ORDER BY Preco DESC
LIMIT 5;

Obs: A sintaxe pode variar ligeiramente (ex: TOP no SQL Server, FETCH FIRST no Oracle mais recente).
