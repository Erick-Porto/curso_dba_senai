O que é um Banco de Dados Relacional?

Uma coleção organizada de dados estruturados em tabelas (relações), com colunas (atributos) e linhas (registros/tuplas).

As tabelas podem se relacionar entre si (ex: cliente faz pedidos, pedido contém produtos).

Analogia: Um arquivo super organizado com gavetas (BD), pastas (tabelas), fichas (linhas) e campos (colunas).

[ Clientes | | Pedidos ]
(Tabelas com Linhas, Colunas e Relações)
