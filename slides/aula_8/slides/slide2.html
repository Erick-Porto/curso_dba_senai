Objetivos da Aula

  • Compreender o que são dados e bancos de dados relacionais.
  • Entender o propósito e a estrutura básica da linguagem SQL.
  • Aprender a selecionar dados de tabelas usando SELECT e FROM.
  • Iniciar a filtragem de dados com a cláusula WHERE.
  • Escrever suas primeiras consultas SQL!
