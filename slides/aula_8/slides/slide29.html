Boas Práticas Iniciais em SQL

Para escrever SQL mais legível e fácil de manter:

  • Capitalização: Use MAIÚSCULAS para palavras-chave SQL (SELECT, FROM, WHERE...) e minúsculas (ou o padrão da sua equipe) para nomes de tabelas/colunas.
    SELECT NomeCliente FROM Clientes WHERE Idade > 18;
  • Indentação: Quebre consultas longas em várias linhas, indentando as cláusulas para clareza visual. 
    
SELECT
    Nome,
    Email,
    Cidade
FROM
    Clientes
WHERE
    Idade > 18
    AND Cidade = 'Porto Alegre'
ORDER BY
    Nome ASC;
  • Comentários: Use -- para comentários de linha única. Use /* ... */ para comentários de múltiplas linhas (pode variar). Explique partes complexas ou a intenção da query. 
    
-- Seleciona clientes maiores de idade de POA
SELECT Nome, Email, Cidade
FROM Clientes
WHERE Idade > 18 -- Filtro de idade
  AND Cidade = 'Porto Alegre'; /* Filtro de cidade */
