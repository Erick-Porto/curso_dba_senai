vs SQL Padrão vs. Dialetos

Existe um SQL Padrão (definido por órgãos como ANSI/ISO) que estabelece a base da linguagem.

No entanto, cada SGBD (Sistema Gerenciador de Banco de Dados) implementa seu próprio dialeto, com extensões e variações.

Exemplos de Dialetos:

  • MySQL
  • PostgreSQL (PL/pgSQL)
  • Microsoft SQL Server (T-SQL)
  • Oracle (PL/SQL)

Ponto Chave: A sintaxe do SELECT básico que veremos hoje é muito similar em todos eles!
