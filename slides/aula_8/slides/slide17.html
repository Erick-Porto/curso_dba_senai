Operadores Especiais no WHERE

Operadores úteis para filtros mais complexos:

  • BETWEEN ... AND ...: Verifica se um valor está DENTRO de um intervalo (inclusivo).
    Ex: WHERE Idade BETWEEN 18 AND 25 (equivale a Idade >= 18 AND Idade <= 25)
  • IN (...): Verifica se um valor está EM UMA LISTA de valores possíveis.
    Ex: WHERE Cidade IN ('São Paulo', 'Belo Horizonte', 'Curitiba')
  • LIKE: Busca por PADRÕES em strings (texto). Usa curingas:
    • % : Representa zero ou mais caracteres quaisquer.
    • _ : Representa exatamente UM caractere qualquer.
    Ex: WHERE Nome LIKE 'An%' (começa com "An")
    Ex: WHERE Email LIKE '%@gmail.com' (termina com "@gmail.com")
    Ex: WHERE Codigo LIKE 'PR_00%' (PR + 1 char + 00 + qualquer coisa)
