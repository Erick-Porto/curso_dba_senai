Ordenando Resultados: ORDER BY

Os resultados de um SELECT não têm uma ordem garantida por padrão.

Para classificar as linhas do resultado, usamos a cláusula ORDER BY.

Posição: Geralmente, é a última cláusula principal da consulta SELECT.


SELECT coluna1, coluna2
FROM nome_da_tabela
WHERE condicao
ORDER BY coluna_para_ordenar [ASC | DESC];
  • ASC : Ordem Ascendente (A-Z, 0-9). É o padrão se omitido.
  • DESC : Ordem Descendente (Z-A, 9-0).
