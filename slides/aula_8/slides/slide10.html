A Consulta Fundamental: SELECT e FROM

Toda consulta básica precisa de duas informações essenciais:

  • O QUÊ você quer ver? Usamos SELECT para listar as colunas desejadas.
  • DE ONDE vêm esses dados? Usamos FROM para indicar a tabela de origem.

SELECT coluna1, coluna2 -- O Quê
FROM nome_da_tabela;    -- De Onde
