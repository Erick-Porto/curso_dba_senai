Operadores de Comparação no WHERE

Usados para comparar valores nas colunas:

  • = : Igual a (Ex: WHERE Cidade = 'Rio de Janeiro')
  • != ou <> : Diferente de (Ex: WHERE Status != 'Inativo')
  • < : Menor que (Ex: WHERE Preco < 50.00)
  • > : Maior que (Ex: WHERE Idade > 18)
  • <= : Menor ou igual a (Ex: WHERE Quantidade <= 10)
  • >= : Maior ou igual a (Ex: WHERE Salario >= 2500.00)

Importante: Valores de texto (VARCHAR, TEXT, etc.) geralmente precisam estar entre aspas simples (' '). Números não.
