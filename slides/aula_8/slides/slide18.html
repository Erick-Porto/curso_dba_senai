Demonstração Ao Vivo 3

Usando operadores especiais:

  • Faixa de Idade:
    SELECT Nome, Idade FROM Clientes WHERE Idade BETWEEN 20 AND 30;
  • Cidades Específicas:
    SELECT Nome, Cidade FROM Clientes WHERE Cidade IN ('Recife', 'Salvador');
  • Produtos que começam com 'Camiseta':
    SELECT Nome FROM Produtos WHERE Nome LIKE 'Camiseta%';
  • Emails do Yahoo:
    SELECT Nome, Email FROM Clientes WHERE Email LIKE '%@yahoo.com';
