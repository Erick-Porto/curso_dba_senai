Criando sua Primeira Stored Procedure

Uso do DELIMITER para mudar o terminador de comando padrão (;) em SGBDs como MySQL/MariaDB.


DELIMITER $$ -- Muda o delimitador para $$

CREATE PROCEDURE NomeDaProcedure (
    param1 TIPO_DADO,
    param2 TIPO_DADO
    -- Mais parâmetros aqui...
)
BEGIN
    -- Comandos SQL aqui
    -- Ex: SELECT * FROM SuaTabela WHERE coluna = param1;
END $$

DELIMITER ; -- Restaura o delimitador padrão
  • CREATE PROCEDURE NomeDaProcedure (...): Define o nome e os parâmetros.
  • BEGIN ... END: Bloco que contém o corpo da procedure.
