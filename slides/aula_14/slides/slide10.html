Como Executar uma Stored Procedure

Comando CALL (padrão SQL, MySQL, PostgreSQL):


-- Sem parâmetros
CALL NomeDaProcedure();

-- Com parâmetros IN e OUT (Exemplo MySQL)
CALL NomeDaProcedureComParams(valor_param_in, @variavel_out);
SELECT @variavel_out AS ResultadoOut;

Comando EXECUTE ou EXEC (SQL Server, Oracle - sintaxe pode variar):


-- Exemplo SQL Server
DECLARE @resultado_out TIPO_DADO;
EXEC NomeDaProcedure valor_param_in, @resultado_out OUTPUT;
SELECT @resultado_out;

Exemplo prático (MySQL): Supondo ObterClientePorID(IN id_cliente INT, OUT nome_cliente VARCHAR(100))


CALL ObterClientePorID(1, @nome);
SELECT @nome AS NomeDoClienteEncontrado;
