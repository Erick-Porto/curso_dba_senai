Scripts SQL: Populando com INSERT INTO (Exemplos)

Após criar as estruturas (CREATE TABLE), precisamos de alguns dados para testar e dar vida ao nosso banco!

Ordem de Inserção Importa (Devido às Chaves Estrangeiras):

  1. Primeiro, popule as tabelas que não dependem de outras (ou cujas dependências já estão satisfeitas). Estas são as tabelas "pai" nos relacionamentos.
    • Ex: Editoras, Autores, Generos, Membros.
  2. Depois, popule as tabelas que possuem Chaves Estrangeiras, usando os IDs das tabelas "pai" já inseridos.
    • Ex: Livros (que depende de Editoras).
  3. Em seguida, as tabelas associativas (que dependem de duas ou mais tabelas "pai").
    • Ex: Livro_Autores, Livro_Generos.
  4. Finalmente, tabelas que podem depender de múltiplas outras.
    • Ex: Emprestimos (que depende de Livros e Membros).

-- Editoras
INSERT INTO Editoras (NomeEditora) VALUES ('Editora Alpha'), ('Companhia das Letras');

-- Autores
INSERT INTO Autores (NomeAutor, Nacionalidade) VALUES
('J.R.R. Tolkien', 'Britânico'), ('Agatha Christie', 'Britânica'), ('Machado de Assis', 'Brasileiro');

-- Generos
INSERT INTO Generos (NomeGenero) VALUES ('Fantasia'), ('Mistério'), ('Romance Brasileiro'), ('Ficção Científica');

-- Livros (Supondo que os IDs das editoras sejam 1, 2, 3 após inserção)
INSERT INTO Livros (Titulo, AnoPublicacao, EditoraID, QtdExemplaresDisponiveis) VALUES
('O Senhor dos Anéis', 1954, 1, 5),
('Assassinato no Expresso do Oriente', 1934, 2, 3),
('Dom Casmurro', 1899, 1, 7); -- Exemplo: Machado pela Editora Alpha

-- Livro_Autores (Obtendo IDs dinamicamente com subqueries é mais robusto)
INSERT INTO Livro_Autores (LivroID, AutorID) VALUES
((SELECT LivroID FROM Livros WHERE Titulo = 'O Senhor dos Anéis'), (SELECT AutorID FROM Autores WHERE NomeAutor = 'J.R.R. Tolkien')),
((SELECT LivroID FROM Livros WHERE Titulo = 'Assassinato no Expresso do Oriente'), (SELECT AutorID FROM Autores WHERE NomeAutor = 'Agatha Christie')),
((SELECT LivroID FROM Livros WHERE Titulo = 'Dom Casmurro'), (SELECT AutorID FROM Autores WHERE NomeAutor = 'Machado de Assis'));
