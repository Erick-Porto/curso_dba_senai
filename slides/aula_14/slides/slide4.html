Conhecendo as Ferramentas: SPs, Funções e Triggers

Stored Procedures (SPs)

Blocos de código SQL nomeados. Podem ter parâmetros (IN, OUT, INOUT). Executam DML/DDL. Invocadas com CALL/EXEC.

Functions (UDFs)

Sempre retornam um valor. Usadas em expressões SQL. Geralmente não modificam dados.

Triggers

Executados AUTOMATICAMENTE em resposta a eventos DML (INSERT, UPDATE, DELETE) em uma tabela.

CALL / EXEC (IN, OUT params) Function (UDF) Retorna Valor (Em SELECTs) Trigger Automático (Evento DML) Objetos de Programação SQL Todos são blocos de código SQL armazenados no banco.
