Objetivos da Aula Prática

  • Aplicar o processo completo de modelagem de dados a partir de um estudo de caso realista.
  • Realizar o levantamento de requisitos e identificar entidades, atributos e relacionamentos.
  • Desenvolver o Modelo Entidade-Relacionamento (MER/DER) conceitual.
  • Converter o MER para o Modelo Lógico Relacional (tabelas, PKs, FKs).
  • Aplicar princípios básicos de Normalização (até 3FN).
  • Implementar o Modelo Físico, gerando scripts SQL CREATE TABLE e INSERT.
  • Elaborar a documentação essencial do projeto (DER, Dicionário de Dados).
  • Adotar boas práticas e padrões profissionais em todo o processo.
