DEMO: Stored Procedure ListarClientes

Cenário: Criar uma SP para listar todos os clientes.

DER (Simplificado):

CodCli (PK) Nome, Email...

Código SQL:


DELIMITER $$
CREATE PROCEDURE ListarClientes()
BEGIN
    SELECT CodCli, Nome, Email FROM Clientes ORDER BY Nome;
END $$
DELIMITER ;

Chamando a Procedure:


CALL ListarClientes();

Demonstração ao Vivo: Criar e executar.
