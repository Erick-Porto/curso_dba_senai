Exercício 1: Backup Completo Lógico com mysqldump

Objetivo: Realizar um backup lógico completo de um banco de dados de exemplo.

Banco de Dados de Exemplo: empresa_simples (a ser criado).


-- Script DDL para criar o banco e tabelas (executar no MySQL)
CREATE DATABASE IF NOT EXISTS empresa_simples;
USE empresa_simples;

CREATE TABLE departamentos (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nome_depto VARCHAR(50) NOT NULL UNIQUE
);

CREATE TABLE funcionarios (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nome VARCHAR(100) NOT NULL,
    cargo VARCHAR(50),
    data_admissao DATE,
    id_depto INT,
    FOREIGN KEY (id_depto) REFERENCES departamentos(id)
);

-- Script DML para popular (executar no MySQL)
INSERT INTO departamentos (nome_depto) VALUES ('RH'), ('TI'), ('Vendas');
INSERT INTO funcionarios (nome, cargo, data_admissao, id_depto) VALUES
('Ana Silva', 'Gerente de RH', '2020-01-15', 1),
('Bruno Costa', 'Desenvolvedor Pleno', '2021-06-01', 2),
('Carla Lima', 'Analista de Vendas', '2022-03-10', 3),
('Daniel Moreira', 'Estagiário TI', '2023-01-20', 2);

Tarefas:

  1. No seu cliente MySQL (linha de comando ou Workbench), execute os scripts DDL e DML acima para criar e popular o banco empresa_simples. Verifique se as tabelas e dados foram criados.
  2. Abra o terminal do seu sistema operacional (Bash no Linux/macOS ou CMD/PowerShell no Windows).
  3. Use o comando mysqldump para realizar um backup lógico completo do banco empresa_simples.
    • Use um usuário MySQL com as permissões necessárias (ex: seu usuário root local para este exercício, ou um usuário de backup).
    • Inclua as opções: --single-transaction, --routines, --triggers, --events (mesmo que não tenhamos criado routines/triggers/events neste exemplo, é boa prática incluí-los).
    • Redirecione a saída para um arquivo chamado empresa_simples_backup_COMPLETO.sql.
  4. Verifique se o arquivo empresa_simples_backup_COMPLETO.sql foi criado no diretório onde você executou o comando (ou no caminho especificado).
  5. (Opcional) Abra o arquivo SQL com um editor de texto e observe seu conteúdo (CREATE TABLE, INSERT INTO).
