Exercício 2: Restaurar Backup Lógico em Banco Limpo

Objetivo: Praticar a restauração de um backup lógico em um novo banco de dados.

Pré-requisito: Ter completado o Exercício 1 e ter o arquivo empresa_simples_backup_COMPLETO.sql.

Tarefas:

  1. Simular um "Desastre":
    • No seu cliente MySQL, execute o comando: DROP DATABASE empresa_simples;
    • Confirme que o banco empresa_simples não existe mais (ex: com SHOW DATABASES;).
  2. Preparar para a Restauração:
    • Crie um novo banco de dados vazio onde o backup será restaurado. Vamos chamá-lo de empresa_restaurada: 
      CREATE DATABASE empresa_restaurada;
  3. Restaurar o Backup:
    • No terminal do seu sistema operacional (onde você tem o arquivo empresa_simples_backup_COMPLETO.sql), use o cliente mysql para importar o arquivo de backup para o banco empresa_restaurada.
    • O comando será algo como: 
      mysql -u seu_usuario -pSuaSenha empresa_restaurada < empresa_simples_backup_COMPLETO.sql
      (Lembre-se de ajustar seu_usuario e SuaSenha. Se o arquivo estiver comprimido, você precisará descomprimi-lo primeiro ou usar um pipe com gunzip -c).
  4. Verificar a Restauração:
    • Conecte-se ao banco empresa_restaurada via cliente MySQL ou MySQL Workbench.
    • Execute SHOW TABLES; para ver se as tabelas foram criadas.
    • Execute SELECT * FROM funcionarios; e SELECT * FROM departamentos; para confirmar que os dados foram restaurados.
