O Pergaminho Base: O Mundo Fraturado

Para começar sua jornada, você recebe o "Pergaminho Base" (o script aventura_modificado.sql). Este script contém a estrutura de RPG_Mundo como ela é agora.

Note as seguintes alterações feitas pelos Mestres do Jogo:

  • A tabela Monstros agora possui uma coluna StatusMonstro VARCHAR(20) DEFAULT 'Ativo'. Este será o "interruptor" para nossas magias automáticas.
  • Um novo TipoItemID foi criado: 'Material de Crafting'.
  • Novos Itens foram adicionados, como 'Fragmento de Sombra', 'Essência Elemental' e 'Couro de Besta'. Atualmente, ninguém os possui.

O GRANDE PROBLEMA: Não existe NENHUMA ligação entre os monstros e os novos materiais! Não há como saber o que um "Lobo Cinzento" deixa cair ao ser derrotado. Além disso, não há como combinar esses itens.

Seu primeiro dever é compreender o mundo como ele é, para então poder expandi-lo.
