O Pergaminho da Criação: RPG_Mundo

Antes de manipularmos os dados, bravos escribas, precisamos entender a estrutura do nosso reino, RPG_Mundo. O script SQL a seguir é o mapa ancestral que define as terras, criaturas, heróis e magias.

Este pergaminho contém os comandos CREATE DATABASE e CREATE TABLE, que são parte da DDL (Data Definition Language) - a linguagem dos Arquitetos Divinos - e também comandos INSERT iniciais para popular nosso mundo com suas primeiras lendas.

Vocês podem copiar e executar este script em seu SGBD (MySQL Workbench ou similar) para ter seu próprio reino de RPG_Mundo para praticar os feitiços DML!


-- =============================================
-- Script para Criação do Banco de Dados RPG_Mundo
-- (Pergaminho dos Arquitetos)
-- =============================================

-- 1. CRIAR O REINO DIGITAL (SE NÃO EXISTIR)
CREATE DATABASE IF NOT EXISTS RPG_Mundo
CHARACTER SET utf8mb4
COLLATE utf8mb4_unicode_ci;

-- 2. SINTONIZAR-SE COM O REINO
USE RPG_Mundo;

-- 3. ERGUER AS ESTRUTURAS (Conteúdo Omitido por Brevidade - Veja o script completo)
-- Tabela de Racas (Linhagens Ancestrais)
CREATE TABLE IF NOT EXISTS Racas ( RacaID INT AUTO_INCREMENT PRIMARY KEY, NomeRaca VARCHAR(50) NOT NULL UNIQUE, /*...*/ );
-- Tabela de Classes (Caminhos do Herói)
CREATE TABLE IF NOT EXISTS Classes ( ClasseID INT AUTO_INCREMENT PRIMARY KEY, NomeClasse VARCHAR(50) NOT NULL UNIQUE, /*...*/ );
-- Tabela de Personagens (Lendas em Formação)
CREATE TABLE IF NOT EXISTS Personagens ( PersonagemID INT AUTO_INCREMENT PRIMARY KEY, NomePersonagem VARCHAR(100) NOT NULL, /*...*/ );
-- ... (demais tábuas sagradas: TiposMagia, Magias, PersonagemMagias, TiposMonstro, Monstros, MonstroMagias, Cenarios, CenarioMonstros, TiposItem, Itens, Inventarios, InventarioItens)

-- 4. INVOCAR OS PRIMEIROS HABITANTES (Conteúdo Omitido por Brevidade)
INSERT INTO Racas (NomeRaca, DescricaoRaca, /*...*/) VALUES ('Humano', /*...*/);
INSERT INTO Classes (NomeClasse, DescricaoClasse, /*...*/) VALUES ('Guerreiro', /*...*/);
-- ... (demais invocações)

-- =============================================
-- FIM DO PERGAMINHO DOS ARQUITETOS
-- =============================================
SELECT 'O Reino de RPG_Mundo foi forjado e as primeiras lendas despertaram!' AS StatusDaCriacao;
