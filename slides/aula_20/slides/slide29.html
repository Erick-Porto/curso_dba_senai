Dica Extra: GitHub para DBAs (Se Aplicável)

Para profissionais de tecnologia, o GitHub não é apenas para desenvolvedores de software. DBAs também podem usá-lo para construir um portfólio prático!

O que um DBA Júnior pode colocar no GitHub?

  • Scripts SQL do Projeto Final:
    • Seu script DDL (CREATE TABLE, constraints, etc.).
    • Seu script DML (INSERTs para carga inicial de dados de exemplo).
    • Exemplos de Stored Procedures, Triggers ou Functions que você criou para o projeto.
  • Scripts de Administração (Exemplos):
    • Se você criou scripts para tarefas de backup simples (como no exercício da aula anterior), monitoramento básico, ou outras rotinas de administração, pode compartilhá-los.
  • Modelagem de Dados:
    • Uma imagem do seu Diagrama Entidade-Relacionamento (DER Chen).
    • O script de criação do banco gerado por uma ferramenta de modelagem (se usou).
  • Documentação do Projeto: Um arquivo README.md bem escrito no seu repositório explicando cada projeto, os desafios e as soluções implementadas.

Benefícios de Usar o GitHub:

  • Portfólio Prático: Demonstra suas habilidades de forma tangível, além do que está no currículo.
  • Mostra Iniciativa: Indica que você vai além do básico e se dedica a projetos.
  • Facilita a Avaliação Técnica: Recrutadores técnicos ou gestores podem revisar seu código e entender melhor seu nível.
  • Aprendizado de Controle de Versão: Usar Git e GitHub é uma habilidade valiosa em qualquer área de TI.

Crie repositórios públicos para seus projetos e adicione o link do seu perfil GitHub no seu LinkedIn e currículo!
