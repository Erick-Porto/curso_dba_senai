Demonstração Ao Vivo 4: Criando e Usando Views

Vamos criar e consultar nossa visão:

  1. Criar a View: 
    -- Criar view para deptos da Sede (ajuste conforme seus dados)
CREATE OR REPLACE VIEW vw_DepartamentosSede AS
SELECT DepartamentoID, NomeDepto, Sigla -- Omitindo DataCriacao e CentroCusto
FROM Departamentos
WHERE Localizacao = 'Sede'; -- Assumindo que 'Sede' existe
  2. Consultar a View: Trate-a como uma tabela. 
    SELECT * FROM vw_DepartamentosSede;
SELECT NomeDepto FROM vw_DepartamentosSede ORDER BY NomeDepto;
  3. Ver o Código da View:
    • MySQL: SHOW CREATE VIEW vw_DepartamentosSede;
    • PostgreSQL: \d+ vw_DepartamentosSede ou consultar pg_views.
  4. Demonstrar Dependência: Atualize a Localizacao de um departamento na tabela base Departamentos (para algo diferente de 'Sede') e mostre que ele some da view na próxima consulta.
  5. (Opcional) Tentar um UPDATE simples na view (se aplicável e permitido pelo SGBD).
