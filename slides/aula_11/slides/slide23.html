Views: Sintaxe e Criação

Para criar uma view, usamos o comando CREATE VIEW.

Sintaxe Básica:


CREATE [OR REPLACE] VIEW nome_da_view [(coluna1, coluna2, ...)] -- OR REPLACE atualiza se já existir; Nomes de coluna opcionais
AS
SELECT coluna_tabela1, coluna_tabela2, calculo_xyz -- A consulta SELECT que define a view
FROM tabela_base1
[JOIN tabela_base2 ON ...]
[WHERE condicao]
[GROUP BY ...]
[ORDER BY ...]; -- ORDER BY em views tem efeito limitado ou nulo em alguns SGBDs

Exemplo Simples: Criar uma view para mostrar apenas departamentos ativos (se tivéssemos uma coluna 'Ativo').


-- Supondo que Departamentos tenha uma coluna 'Ativo' BOOLEAN
CREATE VIEW vw_DepartamentosAtivos AS
SELECT DepartamentoID, NomeDepto, Localizacao
FROM Departamentos
WHERE Ativo = TRUE;

Consultando a View: Depois de criada, usamos como uma tabela normal.

SELECT * FROM vw_DepartamentosAtivos WHERE Localizacao = 'Sede';
