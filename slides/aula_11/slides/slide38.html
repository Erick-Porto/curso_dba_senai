Hora de Construir: Exercícios Práticos DDL

Vamos colocar em prática os comandos CREATE, ALTER e as boas práticas que aprendemos.

Ambiente: Use um banco de dados de teste seguro onde você tenha permissão para criar, alterar e remover tabelas.

Teremos 3 exercícios com dificuldade gradual:

  1. Criar uma tabela básica com constraints.
  2. Alterar uma tabela existente, adicionando elementos.
  3. Criar tabelas relacionadas e uma view.

Preparem suas ferramentas de construção!
