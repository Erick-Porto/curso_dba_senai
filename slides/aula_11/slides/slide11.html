Demonstração Ao Vivo 1: Criando Nossa Primeira Tabela

Vamos ao SGBD (MySQL/PostgreSQL)!

  1. Conectar ao banco de dados de teste.
  2. Executar o comando CREATE TABLE Departamentos (escolha a versão do seu SGBD).
  3. Verificar se a tabela foi criada com sucesso (listando tabelas).
  4. Examinar a estrutura da tabela criada:
    • MySQL: DESCRIBE Departamentos; ou SHOW CREATE TABLE Departamentos;
    • PostgreSQL: \d Departamentos
    (Observar os tipos de dados, se aceita NULL, as chaves e defaults).
  5. Tentar inserir um departamento sem nome (NOT NULL) e ver o erro.
  6. Tentar inserir dois departamentos com o mesmo nome (UNIQUE) e ver o erro.
