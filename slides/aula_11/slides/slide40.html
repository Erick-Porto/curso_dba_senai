Exercício 1 - Solução Comentada

Código para criar a tabela Produtos:

Solução (MySQL):


CREATE TABLE Produtos (
    ProdutoID INT AUTO_INCREMENT PRIMARY KEY, -- PK e Auto-incremento
    NomeProduto VARCHAR(100) NOT NULL,       -- Obrigatório
    CodigoBarras VARCHAR(13) UNIQUE,          -- Único (permite NULL)
    PrecoCusto DECIMAL(10, 2) NOT NULL,      -- Obrigatório, precisão monetária
    DataCadastro DATE DEFAULT CURRENT_DATE   -- Data padrão
);

Solução (PostgreSQL):


CREATE TABLE Produtos (
    ProdutoID SERIAL PRIMARY KEY,              -- PK e Auto-incremento (SERIAL)
    NomeProduto VARCHAR(100) NOT NULL,
    CodigoBarras VARCHAR(13) UNIQUE,
    PrecoCusto DECIMAL(10, 2) NOT NULL,
    DataCadastro DATE DEFAULT CURRENT_DATE
);

Verificação (Exemplo MySQL): DESCRIBE Produtos;

Verificação (Exemplo PostgreSQL): \d Produtos

Confirme se os tipos, nullability, chaves e defaults estão corretos.
