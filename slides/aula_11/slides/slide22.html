Views (Visões): O que são?

Uma View (ou Visão) é um objeto do banco de dados que consiste em uma consulta SELECT armazenada.

Ela se comporta como uma tabela virtual: você pode consultá-la (SELECT * FROM nome_da_view), mas ela geralmente não armazena dados fisicamente.

Quando você consulta uma view, o SGBD executa a consulta SELECT subjacente que a define.

Analogia: Uma "janela" personalizada que mostra apenas partes específicas de um ou mais "cômodos" (tabelas), ou uma combinação deles.
