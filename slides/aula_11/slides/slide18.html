Demonstração Ao Vivo 2: Modificando a Tabela `Departamentos`

Vamos aplicar alterações na nossa tabela:

  1. Adicionar Coluna: ADD COLUMN Localizacao VARCHAR(100);
  2. Verificar: DESCRIBE Departamentos; / \d Departamentos
  3. Modificar Coluna: MODIFY COLUMN NomeDepto VARCHAR(100) NOT NULL; (MySQL) ou ALTER COLUMN NomeDepto TYPE VARCHAR(100); (PgSQL).
  4. Verificar: DESCRIBE Departamentos; / \d Departamentos
  5. Adicionar Constraint CHECK: ADD CONSTRAINT chk_localizacao CHECK (Localizacao IN ('Sede', 'Filial Norte', 'Filial Sul')); (Ajuste a sintaxe do CHECK se necessário para seu SGBD).
  6. Verificar Constraints: SHOW CREATE TABLE Departamentos; (MySQL) / \d Departamentos (PgSQL).
  7. (Opcional) Tentar inserir/atualizar violando o CHECK.
  8. (Opcional) Remover a constraint: DROP CONSTRAINT chk_localizacao;.
