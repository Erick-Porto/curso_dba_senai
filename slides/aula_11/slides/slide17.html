/ ALTER TABLE: Adicionar/Remover Constraints

Podemos adicionar ou remover constraints (além das definidas na criação) usando ALTER TABLE.

Adicionar Constraint:


-- Adicionar UNIQUE constraint na coluna Email
ALTER TABLE Clientes
ADD CONSTRAINT uq_cliente_email UNIQUE (Email);

-- Adicionar FOREIGN KEY
ALTER TABLE Pedidos
ADD CONSTRAINT fk_pedido_cliente
    FOREIGN KEY (ClienteID) REFERENCES Clientes(ClienteID);

-- Adicionar CHECK constraint
ALTER TABLE Produtos
ADD CONSTRAINT chk_estoque_nao_negativo CHECK (Estoque >= 0);

Remover Constraint: (Sintaxe pode variar)


-- MySQL
ALTER TABLE Clientes DROP CONSTRAINT uq_cliente_email;
ALTER TABLE Pedidos DROP FOREIGN KEY fk_pedido_cliente; -- Sintaxe especial para FK
ALTER TABLE Produtos DROP CHECK chk_estoque_nao_negativo; -- Sintaxe especial para CHECK

-- PostgreSQL
ALTER TABLE Clientes DROP CONSTRAINT uq_cliente_email;
ALTER TABLE Pedidos DROP CONSTRAINT fk_pedido_cliente;
ALTER TABLE Produtos DROP CONSTRAINT chk_estoque_nao_negativo;

É importante dar nomes explícitos às constraints (CONSTRAINT nome_constraint ...) para facilitar a remoção posterior.
