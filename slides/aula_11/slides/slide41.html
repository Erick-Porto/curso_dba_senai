Exercício 2 (Intermediário): Modificando `Produtos`

Sua Missão: Refinar e melhorar a estrutura da tabela Produtos existente.

Tarefas (usando ALTER TABLE):

  1. Adicione uma nova coluna chamada EstoqueAtual do tipo Inteiro (INT). Ela não pode ser nula e seu valor padrão deve ser 0.
  2. Adicione uma constraint CHECK chamada chk_preco_positivo para garantir que a coluna PrecoCusto contenha sempre um valor maior que zero (> 0).
  3. Crie um índice chamado idx_produto_nome na coluna NomeProduto para otimizar futuras buscas por nome.
  4. Modifique a coluna CodigoBarras para permitir um tamanho maior, alterando seu tipo para VARCHAR(14) (mantendo a constraint UNIQUE).

Dica: Execute um comando ALTER TABLE para cada tarefa. Lembre-se que a sintaxe de MODIFY/ALTER COLUMN e ADD CONSTRAINT pode variar um pouco entre MySQL e PostgreSQL. Verifique a estrutura após cada alteração.
