Demonstração Ao Vivo 5: Criando um Índice

Vamos otimizar a busca por localização:

  1. Criar o Índice: 
    -- Criar índice na coluna Localizacao da tabela Departamentos
CREATE INDEX idx_dept_localizacao ON Departamentos (Localizacao);
  2. Verificar Criação:
    • MySQL: SHOW INDEX FROM Departamentos;
    • PostgreSQL: \di ou \d Departamentos (mostra índices associados).
  3. (Opcional - Conceitual) Discutir `EXPLAIN`: 
    -- Explicar como analisar a saída (uso do índice vs. table scan)
EXPLAIN SELECT * FROM Departamentos WHERE Localizacao = 'Sede';

    (Nota: A interpretação do EXPLAIN é um tópico avançado, mas vale a pena mencionar que é a ferramenta para verificar se um índice está sendo usado).

  4. Remover o Índice (se necessário): 
    DROP INDEX idx_dept_localizacao ON Departamentos; -- MySQL
-- DROP INDEX idx_dept_localizacao; -- PostgreSQL
