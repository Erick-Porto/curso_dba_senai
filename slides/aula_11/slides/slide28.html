Índices: Sintaxe e Criação

Para criar um índice explicitamente (além dos criados por PK/UNIQUE), usamos CREATE INDEX.

Sintaxe Básica:


CREATE INDEX nome_do_indice -- Dê um nome descritivo (ex: idx_tabela_coluna)
ON nome_da_tabela (coluna1 [ASC | DESC], coluna2 [ASC | DESC], ...);

Exemplo: Criar um índice na coluna NomeDepto da tabela Departamentos para acelerar buscas por nome.


CREATE INDEX idx_departamentos_nome ON Departamentos (NomeDepto);

Exemplo: Índice Composto (em múltiplas colunas - a ordem importa!). Útil para filtros que usam ambas as colunas.


-- Para otimizar WHERE CategoriaID = ? AND Preco > ?
CREATE INDEX idx_produtos_cat_preco ON Produtos (CategoriaID, Preco);

Exemplo: Índice Descendente (Pode ajudar em ORDER BY ... DESC).


CREATE INDEX idx_pedidos_data_desc ON Pedidos (DataPedido DESC);
