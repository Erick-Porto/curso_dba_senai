Recapitulação Rápida H1

Fundamentos Estabelecidos:

  • DDL é para definir a estrutura do banco.
  • CREATE TABLE é o comando para criar novas tabelas.
  • Escolher Tipos de Dados corretos é crucial (INT, VARCHAR, DATE, DECIMAL...).
  • Constraints iniciais (PRIMARY KEY, NOT NULL, UNIQUE) garantem integridade básica.
  • Boas práticas de nomenclatura e definição ajudam na manutenção futura.
