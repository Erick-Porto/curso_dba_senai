ALTER TABLE: Adicionar Coluna (ADD COLUMN)

Para adicionar uma nova coluna a uma tabela existente:

Sintaxe:


ALTER TABLE nome_da_tabela
ADD COLUMN nome_nova_coluna TIPO_DE_DADO [DEFAULT valor_padrao] [CONSTRAINTS];

Exemplo: Adicionar uma coluna `Localizacao` à nossa tabela `Departamentos`.


ALTER TABLE Departamentos
ADD COLUMN Localizacao VARCHAR(100) DEFAULT 'Indefinido';

Considerações:

  • Se a coluna for `NOT NULL` sem `DEFAULT`, a operação pode falhar em tabelas com dados (ou exigir um valor padrão inicial).
  • Pode ser uma operação rápida ou lenta dependendo do SGBD e do tamanho da tabela.
