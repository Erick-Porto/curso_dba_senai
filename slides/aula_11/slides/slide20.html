Demonstração Ao Vivo 3: Usando (e Temendo) o DROP TABLE

Vamos ver o poder destrutivo (em ambiente seguro!):

  1. Criar Tabela de Teste: 
    CREATE TABLE TabelaParaDropar (id INT PRIMARY KEY, nome VARCHAR(10));
  2. Inserir Dados: 
    INSERT INTO TabelaParaDropar VALUES (1, 'Teste');
  3. Verificar Existência: 
    SELECT * FROM TabelaParaDropar;
-- Listar tabelas para confirmar que ela existe.
  4. Executar o DROP (com confirmação verbal!): 
    DROP TABLE TabelaParaDropar;
  5. Verificar Desaparecimento: 
    -- Tentar SELECT (deve dar erro "tabela não encontrada")
SELECT * FROM TabelaParaDropar;
-- Listar tabelas novamente (não deve aparecer).

Enfatizar a natureza permanente e a necessidade de cautela extrema.
