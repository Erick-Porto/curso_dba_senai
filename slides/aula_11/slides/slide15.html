ALTER TABLE: Modificar Coluna (MODIFY/ALTER COLUMN)

Para alterar as propriedades de uma coluna existente (tipo, tamanho, nullability, default, etc.). A sintaxe varia bastante!

Sintaxe (Exemplos):


-- MySQL: Alterar tipo e adicionar NOT NULL
ALTER TABLE Departamentos
MODIFY COLUMN NomeDepto VARCHAR(100) NOT NULL;

-- MySQL: Adicionar valor DEFAULT
ALTER TABLE Departamentos
ALTER COLUMN Localizacao SET DEFAULT 'Sede';

-- PostgreSQL: Alterar tipo
ALTER TABLE Departamentos
ALTER COLUMN NomeDepto TYPE VARCHAR(100);

-- PostgreSQL: Adicionar NOT NULL
ALTER TABLE Departamentos
ALTER COLUMN Sigla SET NOT NULL;

-- PostgreSQL: Remover DEFAULT
ALTER TABLE Departamentos
ALTER COLUMN DataCriacao DROP DEFAULT;

CUIDADO EXTREMO:

  • Alterar o tipo de dados (ex: `VARCHAR` para `INT`) pode causar perda de dados se a conversão não for possível.
  • Diminuir o tamanho de um `VARCHAR` ou `DECIMAL` pode truncar dados existentes.
  • Adicionar NOT NULL a uma coluna com `NULL`s existentes falhará (precisa tratar os `NULL`s primeiro).

SEMPRE TESTE extensivamente antes de aplicar em produção!
