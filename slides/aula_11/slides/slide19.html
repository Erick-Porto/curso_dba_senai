Comando DROP TABLE

Este comando é usado para remover permanentemente uma tabela inteira do banco de dados.

Sintaxe:


DROP TABLE nome_da_tabela;

-- Forma mais segura para scripts (evita erro se a tabela não existir)
DROP TABLE IF EXISTS nome_da_tabela;

PERIGO MÁXIMO! IRREVERSÍVEL!

DROP TABLE apaga a estrutura da tabela E TODOS OS DADOS contidos nela. Não há "desfazer" fácil (somente restauração de backup).

Boas Práticas Essenciais:

  • Verifique Dependências: Outras tabelas (via FKs), Views, Procedures, Functions dependem desta tabela? Remova ou ajuste as dependências primeiro.
  • Confirme o Nome: Tenha 1000% de certeza que está digitando o nome correto da tabela a ser removida.
  • BACKUP! BACKUP! BACKUP! Garanta que existe um backup recente e testado antes de executar qualquer DROP em produção.
  • Permissões Restritas: Em produção, o privilégio de executar DROP deve ser muito restrito.
