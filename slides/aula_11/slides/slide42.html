Exercício 2 - Solução Comentada

Comandos para alterar a tabela Produtos:

Solução (MySQL):


-- 1. Adicionar Coluna EstoqueAtual
ALTER TABLE Produtos
ADD COLUMN EstoqueAtual INT NOT NULL DEFAULT 0;

-- 2. Adicionar Constraint CHECK
ALTER TABLE Produtos
ADD CONSTRAINT chk_preco_positivo CHECK (PrecoCusto > 0);

-- 3. Criar Índice
CREATE INDEX idx_produto_nome ON Produtos (NomeProduto);

-- 4. Modificar Coluna CodigoBarras
ALTER TABLE Produtos
MODIFY COLUMN CodigoBarras VARCHAR(14);

Solução (PostgreSQL):


-- 1. Adicionar Coluna EstoqueAtual
ALTER TABLE Produtos
ADD COLUMN EstoqueAtual INT NOT NULL DEFAULT 0;

-- 2. Adicionar Constraint CHECK
ALTER TABLE Produtos
ADD CONSTRAINT chk_preco_positivo CHECK (PrecoCusto > 0);

-- 3. Criar Índice
CREATE INDEX idx_produto_nome ON Produtos (NomeProduto);

-- 4. Modificar Coluna CodigoBarras
ALTER TABLE Produtos
ALTER COLUMN CodigoBarras TYPE VARCHAR(14);

Verificação: Use DESCRIBE Produtos; (MySQL) ou \d Produtos (PostgreSQL) para ver a coluna adicionada, o tipo alterado e as novas constraints/índices.
