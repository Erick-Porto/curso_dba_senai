Comando CREATE TABLE

Este é o comando fundamental da DDL, usado para criar uma nova tabela vazia no banco de dados.

É aqui que definimos o "esqueleto" onde os dados (manipulados pelo DML) serão armazenados.

Sintaxe Básica:


CREATE TABLE nome_da_tabela (
    nome_coluna1 TIPO_DE_DADO [CONSTRAINTS],
    nome_coluna2 TIPO_DE_DADO [CONSTRAINTS],
    ...
    nome_colunaN TIPO_DE_DADO [CONSTRAINTS]
    -- , CONSTRAINT nome_constraint TIPO (coluna) -- Constraints de tabela (opcional)
);

Dentro dos parênteses, listamos cada coluna, seu tipo de dado e as restrições (constraints) que se aplicam a ela.
