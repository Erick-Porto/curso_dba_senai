Trabalho Final da Disciplina: Desenvolvimento de BD (Parte 3/3)

4. Testes e Funcionamento do Banco de Dados

  • Todas as tabelas deverão estar populadas com dados de teste consistentes e suficientes para a demonstração.
  • Serão realizadas consultas (SELECT) que deverão retornar os resultados esperados, validando a lógica.
  • É imprescindível que as operações de manipulação de dados (INSERT, UPDATE, DELETE) funcionem corretamente, mantendo a integridade referencial.
  • Para fins de avaliação da aprendizagem, deverá ser possível realizar a remoção de registros, demonstrando o tratamento de restrições.
  • Deverá haver o uso de consultas com diferentes tipos de JOINs.
  • Para consultas complexas com JOINs que sejam frequentemente utilizadas, deverão ser criadas VIEWs correspondentes.

5. Apresentação Final

  • O banco de dados desenvolvido deverá ser apresentado em pleno funcionamento.
  • Deverá ser apresentada a utilidade do banco de dados no contexto do tema escolhido, destacando os benefícios que sua implementação traria para a organização ou sistema em questão.
