Demo 4: Transações Simples (COMMIT/ROLLBACK)

Vendo a mágica acontecer (em tabelas de teste):

  1. Cenário COMMIT: 
    BEGIN;
INSERT INTO Categorias_Teste (NomeCategoria) VALUES ('Temporária');
SELECT * FROM Categorias_Teste WHERE NomeCategoria = 'Temporária'; -- Deve aparecer
COMMIT;
SELECT * FROM Categorias_Teste WHERE NomeCategoria = 'Temporária'; -- Deve continuar lá
  2. Cenário ROLLBACK: 
    BEGIN;
UPDATE Produtos_Teste SET Preco = 0.01 WHERE CategoriaID = 1; -- Erro!
SELECT * FROM Produtos_Teste WHERE CategoriaID = 1 AND Preco = 0.01; -- Alteração visível DENTRO da Tx
ROLLBACK;
SELECT * FROM Produtos_Teste WHERE CategoriaID = 1 AND Preco = 0.01; -- Não deve retornar nada! Preços originais restaurados.

As transações nos dão controle sobre quando as alterações se tornam permanentes.
