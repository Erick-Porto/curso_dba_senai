UPDATE: A Cláusula WHERE - NÃO NEGOCIÁVEL!

Esta é, talvez, a parte MAIS IMPORTANTE do comando UPDATE.

A cláusula WHERE funciona exatamente como no SELECT: ela filtra e seleciona QUAIS LINHAS serão afetadas pela alteração definida no SET.


-- Atualizar APENAS o cliente com ID 50
UPDATE Clientes SET Cidade = 'Nova Cidade' WHERE ClienteID = 50;

-- Atualizar APENAS produtos inativos de uma categoria específica
UPDATE Produtos SET Estoque = 0 WHERE Ativo = FALSE AND CategoriaID = 3;

Se você OMITIR a cláusula WHERE...

...O UPDATE será aplicado a TODAS AS LINHAS da tabela!
