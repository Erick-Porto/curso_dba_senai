INSERT: Múltiplas Linhas de Uma Vez

É possível inserir várias linhas com um único comando INSERT, separando os conjuntos de valores por vírgulas.


INSERT INTO Categorias (NomeCategoria) VALUES
  ('Eletrônicos'), -- Primeira linha
  ('Roupas'),      -- Segunda linha
  ('Alimentos');   -- Terceira linha
  

INSERT INTO Produtos (NomeProduto, CategoriaID, Preco, Estoque) VALUES
  ('Smartphone Y', 1, 2500.00, 15), -- Produto 1
  ('Camiseta Polo', 2, 120.00, 50), -- Produto 2
  ('Arroz Integral 1kg', 3, 8.50, 100); -- Produto 3

Vantagem: Geralmente mais performático do que executar múltiplos comandos INSERT individuais, pois reduz a comunicação com o banco de dados.
