Demo 2b: O Desastre do UPDATE sem WHERE

ATENÇÃO: Demonstração em TABELA DE TESTE DESCARTÁVEL! Nunca faça isso em produção!

  1. Verificar Estado Inicial: Contar linhas e ver alguns dados distintos. 
    SELECT COUNT(*) FROM Produtos_Teste_Perigoso;
SELECT DISTINCT CategoriaID FROM Produtos_Teste_Perigoso;
  2. Executar o UPDATE SEM WHERE: 
    -- Ação Perigosa!
UPDATE Produtos_Teste_Perigoso SET CategoriaID = 99; -- Altera a categoria de TODOS!
  3. Verificar o Resultado: Contar linhas (continua igual) e verificar os dados. 
    SELECT COUNT(*) FROM Produtos_Teste_Perigoso;
SELECT DISTINCT CategoriaID FROM Produtos_Teste_Perigoso; -- Deve retornar apenas 99!
SELECT * FROM Produtos_Teste_Perigoso LIMIT 10; -- Ver os dados alterados
