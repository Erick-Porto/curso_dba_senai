DELETE: A Cláusula WHERE - NÃO NEGOCIÁVEL (DE NOVO!)

Assim como no UPDATE, a cláusula WHERE no DELETE é CRÍTICA.

Ela determina QUAIS LINHAS serão permanentemente removidas do banco de dados.


-- Deletar APENAS o pedido com ID 987
DELETE FROM Pedidos WHERE PedidoID = 987;

-- Deletar APENAS logs com mais de 90 dias
DELETE FROM Logs WHERE DataLog < DATE_SUB(NOW(), INTERVAL 90 DAY); -- Exemplo MySQL
-- DELETE FROM Logs WHERE DataLog < NOW() - INTERVAL '90 days'; -- Exemplo PostgreSQL

Se você OMITIR a cláusula WHERE...

...O DELETE será aplicado a TODAS AS LINHAS da tabela!
