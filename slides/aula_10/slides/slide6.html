INSERT INTO: Visão Geral

O comando INSERT é usado para adicionar UMA ou MAIS novas linhas (registros) a uma tabela existente.

Sintaxe Geral:


INSERT INTO nome_da_tabela [(coluna1, coluna2, ...)]
VALUES (valor1, valor2, ...);

Precisamos especificar:

  • A tabela onde inserir (INSERT INTO nome_da_tabela).
  • Opcionalmente, as colunas que receberão valores.
  • Os valores a serem inseridos para essas colunas (VALUES (...)).
