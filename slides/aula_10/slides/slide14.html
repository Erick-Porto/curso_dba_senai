UPDATE: Cláusula SET

A cláusula SET define as alterações a serem feitas.

  • Atualizando uma coluna: 
    SET Email = 'email.corrigido@provider.com'
  • Atualizando múltiplas colunas (separadas por vírgula): 
    SET Preco = 99.90, Estoque = 50, Ativo = TRUE
  • Usando expressões ou o valor atual da coluna: 
    SET Preco = Preco * 1.10 -- Aumenta o preço em 10%
    SET QuantidadeVendida = QuantidadeVendida + 1
    SET NomeCompleto = CONCAT(PrimeiroNome, ' ', Sobrenome) -- Exemplo com função

Os novos valores devem ser compatíveis com os tipos de dados das colunas.
