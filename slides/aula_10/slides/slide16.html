PERIGO! UPDATE sem WHERE!

Executar um comando como este pode causar DANOS IRREPARÁVEIS se não for intencional e feito com extrema cautela (e backup!).


-- EXEMPLO DE DESASTRE POTENCIAL:
UPDATE Usuarios SET Senha = 'senha_padrao_insegura'; -- << SEM WHERE!!
-- ISTO ALTERA A SENHA DE TODOS OS USUÁRIOS DO SISTEMA!
   

-- OUTRO EXEMPLO DE DESASTRE:
UPDATE Produtos SET Preco = 1.00; -- << SEM WHERE!!
-- ISTO MUDA O PREÇO DE TODOS OS PRODUTOS PARA 1.00!

SEMPRE, SEMPRE, SEMPRE verifique sua cláusula WHERE antes de executar um UPDATE!
