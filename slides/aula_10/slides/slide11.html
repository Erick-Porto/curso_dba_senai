Demonstração Ao Vivo 1: Praticando o INSERT

Vamos adicionar dados (usando tabelas de teste!):

  • Inserir uma categoria especificando a coluna NomeCategoria.
  • Tentar inserir omitindo coluna obrigatória (sem default/null). Ver o erro.
  • Inserir um produto omitindo o ProdutoID (assumindo auto-incremento) e usando DEFAULT para `Ativo`.
  • Inserir 2-3 produtos de uma vez usando a sintaxe de múltiplos VALUES.
  • Usar SELECT * FROM ... após cada passo para verificar o resultado.
  • (Opcional) Mostrar como obter o ID gerado (LAST_INSERT_ID() no MySQL, cláusula RETURNING no PostgreSQL).
