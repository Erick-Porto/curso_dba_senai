Ambiente Seguro para Prática

Antes de irmos para os exercícios:

NUNCA pratique comandos DML diretamente em um banco de dados de produção!

Use sempre:

  • Um banco de dados de desenvolvimento ou teste.
  • Cópias das tabelas (ex: CREATE TABLE Tabela_Teste AS SELECT * FROM Tabela_Original;) para poder experimentar sem medo de perder dados importantes.
  • Um usuário com permissões limitadas, se possível.

Para os exercícios de hoje, vamos assumir que vocês criaram as tabelas Categorias_Teste e Produtos_Teste em seu ambiente seguro.
