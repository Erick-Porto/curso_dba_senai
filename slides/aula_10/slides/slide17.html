Demo 2a: UPDATE Seguro com WHERE

A abordagem correta para atualizar dados específicos:

  1. Identificar a(s) linha(s): Use SELECT com uma condição WHERE precisa (idealmente a chave primária). 
    -- Verificar o cliente que quero atualizar (Ex: ID = 15)
SELECT * FROM Clientes_Teste WHERE ClienteID = 15;
  2. Executar o UPDATE: Use a MESMA condição WHERE. 
    -- Corrigir o email do cliente 15
UPDATE Clientes_Teste SET Email = 'email.correto@mail.com' WHERE ClienteID = 15;
  3. Verificar a Alteração: Use SELECT novamente para confirmar que *apenas* a linha desejada foi alterada. 
    SELECT * FROM Clientes_Teste WHERE ClienteID = 15;
