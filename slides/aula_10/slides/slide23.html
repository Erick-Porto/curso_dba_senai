Demo 3a: DELETE Seguro com WHERE

Removendo dados específicos com segurança:

  1. Identificar a(s) linha(s): Use SELECT com a condição WHERE. 
    -- Verificar produtos inativos (Ativo = FALSE) que quero remover
SELECT * FROM Produtos_Teste WHERE Ativo = FALSE;
  2. Executar o DELETE: Use a MESMA condição WHERE. 
    -- Remover os produtos inativos
DELETE FROM Produtos_Teste WHERE Ativo = FALSE;
  3. Verificar a Remoção: Use SELECT novamente para confirmar. 
    -- Deve retornar 0 linhas
SELECT * FROM Produtos_Teste WHERE Ativo = FALSE;
