Recapitulação Rápida H2

Pontos chave sobre UPDATE:

  • UPDATE modifica dados em linhas existentes.
  • A cláusula SET define as colunas e seus novos valores.
  • A cláusula WHERE é ABSOLUTAMENTE ESSENCIAL para especificar quais linhas atualizar.
  • Omitir WHERE no UPDATE é extremamente perigoso.
  • Testar com SELECT antes é uma prática de segurança fundamental.
