Storage Manager em Detalhes

Gerencia a interação com os dados no nível físico (disco) e memória.

[Blocos: Buffer Manager | File Manager | Transaction Manager | Lock Manager]

  • Buffer Manager: Gerencia o cache (buffer pool) na memória principal (RAM) para reduzir acessos a disco (que é lento). Decide quais blocos trazer/remover da memória.
  • File Manager: Gerencia a alocação de espaço em disco e a estrutura dos arquivos que contêm os dados.
  • Transaction Manager: Garante que as transações (conjuntos de operações) sejam atômicas (tudo ou nada) e consistentes, mesmo com falhas ou concorrência.
  • Lock Manager: Gerencia os bloqueios (locks) em objetos do banco (linhas, tabelas) para controlar o acesso concorrente.
