Dados -> Informação -> Banco de Dados

  • Dados: Fatos brutos, isolados, sem contexto.
    • Ex: "Maria", 30, "Rua A, 123", "10/10/2023", 99.50
  • Informação: Dados processados, organizados, com significado e contexto.
    • Ex: A cliente Maria (30 anos), que mora na Rua A, 123, fez uma compra de R$ 99,50 em 10/10/2023.
  • Banco de Dados (BD): Coleção organizada e relacionada de dados, representando informações sobre um domínio específico.
    • Ex: Um conjunto de tabelas (Clientes, Pedidos, Produtos) que armazenam os dados da loja de forma estruturada.
