Dicionário de Dados / Metadados

  • Metadados: São "dados sobre os dados". Descrevem a estrutura do banco de dados.
  • Dicionário de Dados (ou Catálogo do Sistema): É um componente do SGBD que armazena os metadados. É um "mini-banco de dados" sobre o banco de dados principal.

O que armazena?

  • Nomes de tabelas e seus atributos.
  • Tipos de dados de cada atributo (INT, VARCHAR, DATE...).
  • Restrições (Chaves primárias/estrangeiras, NOT NULL, UNIQUE).
  • Informações sobre visões, índices, usuários, permissões.

Importância: Usado pelo SGBD (especialmente Query Processor) para validar e executar consultas. Permite aos usuários e administradores entender a estrutura do BD.
