Setup e Conceito do Dataset

Vamos simular uma mini loja online simples.

Entidades Principais:

  • Categorias: Para agrupar os produtos (Ex: Eletrônicos, Livros, Roupas).
    • Atributos: ID da Categoria, Nome da Categoria.
  • Produtos: Itens à venda na loja (Ex: Smartphone XPTO, Livro "A Arte de SQL", Camiseta Algodão).
    • Atributos: ID do Produto, Nome do Produto, Preço, ID da Categoria (relacionamento).

Objetivo: Criar as tabelas no SGBD (Ex: MySQL/PostgreSQL) usando os comandos SQL que aprendemos.

Pré-requisito: Estar conectado ao banco de dados `minha_loja` criado anteriormente.
