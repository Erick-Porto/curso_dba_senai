Criação: Sintaxe Padrão (ANSI SQL)

A sintaxe mais comum e portável (funciona em MySQL, PostgreSQL, e similar em outros SGBDs) é:

CREATE TEMPORARY TABLE nome_tabela_temporaria (
    coluna1 TIPO_DADO [restrições de coluna],
    coluna2 TIPO_DADO [restrições de coluna],
    ...
    [restrições de tabela, ex: PRIMARY KEY]
);

Exemplo Simples (MySQL/PostgreSQL):

CREATE TEMPORARY TABLE tmp_pedidos_recentes (
    id_pedido INT PRIMARY KEY,
    data_pedido DATE,
    valor_total DECIMAL(10,2)
);

Nota sobre SQL Server: A sintaxe é ligeiramente diferente, usando o prefixo #: 

CREATE TABLE #tmp_pedidos_recentes (
    id_pedido INT PRIMARY KEY,
    data_pedido DATE,
    valor_total DECIMAL(10,2)
);
