H2: Revisão Rápida da Hora 1

Antes de prosseguirmos, vamos recapitular os pontos principais da Hora 1:

  • O que são tabelas temporárias? São "rascunhos" de dados, existindo apenas durante uma sessão.
  • Principais diferenças vs. tabelas permanentes: Persistência (temporárias são destruídas), Visibilidade (temporárias são isoladas por sessão).
  • Três grandes motivos para usar:
    • Armazenar dados intermediários de consultas complexas.
    • Manipular grandes volumes de dados sem impactar tabelas principais.
    • Simplificar a lógica de consultas muito elaboradas.

Com esses conceitos frescos, estamos prontos para explorar os tipos!
