O Que São Tabelas Temporárias?

Tabelas Temporárias são, essencialmente, tabelas "rascunho" que existem apenas durante uma sessão de usuário ou, em alguns casos, uma transação específica.

Elas são armazenadas em uma área temporária do banco de dados (como o tempdb no SQL Server ou um tablespace temporário no Oracle/PostgreSQL) e são invisíveis para outras sessões.

Isso significa que, se dois usuários criarem uma tabela temporária com o mesmo nome, cada um terá sua própria instância isolada, sem interferir no outro.
