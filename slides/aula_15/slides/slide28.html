Exercício 1: Clientes Recentes (Básico)

Objetivo: Criar uma tabela temporária para armazenar informações básicas de clientes de uma cidade específica que se cadastraram no último ano.

Passos:

  1. Crie uma tabela temporária chamada tmp_clientes_recentes_rj com as colunas:
    • id_cliente (INT)
    • nome_cliente (VARCHAR)
    • email_cliente (VARCHAR)
  2. Popule esta tabela com clientes da tabela CLIENTES que atendam aos seguintes critérios:
    • Cidade seja 'Rio de Janeiro'.
    • DataCadastro seja a partir de 01/01/2023 (ou ajuste para "último ano" a partir da data atual).
  3. Consulte todos os dados da tabela tmp_clientes_recentes_rj para verificar o resultado.
  4. (Opcional, mas boa prática) Execute o comando para remover (DROP) a tabela temporária criada.

Mãos à obra!
