Solução Comentada: Exercício 1

Objetivo: Clientes recentes de 'Rio de Janeiro'.

-- Passo 1: Criar a tabela temporária
-- (Exemplo para MySQL/PostgreSQL. Para SQL Server, use: CREATE TABLE #tmp_clientes_recentes_rj)
CREATE TEMPORARY TABLE tmp_clientes_recentes_rj (
    id_cliente INT,
    nome_cliente VARCHAR(255),
    email_cliente VARCHAR(255)
);

-- Passo 2: Popular a tabela com clientes do RJ cadastrados desde 2023-01-01
INSERT INTO tmp_clientes_recentes_rj (id_cliente, nome_cliente, email_cliente)
SELECT
    ID_Cliente,  -- Coluna da tabela CLIENTES
    Nome,        -- Coluna da tabela CLIENTES
    Email        -- Coluna da tabela CLIENTES
FROM CLIENTES
WHERE Cidade = 'Rio de Janeiro'
  AND DataCadastro >= '2023-01-01'; -- Ajuste a data conforme seu SGBD/necessidade

-- Passo 3: Consultar a tabela temporária para verificar
SELECT * FROM tmp_clientes_recentes_rj;

-- Passo 4: (Opcional, mas boa prática) Dropar a tabela
-- DROP TABLE tmp_clientes_recentes_rj;

Comentários:

  • A criação da tabela define a estrutura que queremos para os dados temporários.
  • O INSERT INTO ... SELECT é uma forma eficiente de popular a tabela temporária com base em um filtro na tabela permanente CLIENTES.
  • A consulta final SELECT * apenas verifica o conteúdo da tabela temporária.
